# YaSnippets collection

Comprehensive collection of snippets that have been collected from
many people.

Included modes are:

* clojure-mode
* css-mode
* emacs-lisp-mode
* feature-mode
* haml-mode
* haskell-mode
* html-mode
* rhtml-mode
* java-mode
* js-mode
* js2-mode
* nxml-mode
* ruby-mode
* sass-mode
* sh-mode
